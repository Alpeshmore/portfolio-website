Projects

Line Follower Robot Using ATmgea8

A Line Follower Robot using an ATmega8 microcontroller is an autonomous robotic system designed to follow a predefined path, usually a black or white line on a contrasting surface. It is commonly used in robotics competitions, industrial automation, and educational projects.

An Automatic Boom Barrie Using Arduino

An Automatic Boom Barrier is a motorized gate system that controls vehicle access at entry and exit points, commonly used in parking lots, toll booths, and security checkpoints. It operates using an Arduino microcontroller along with various sensors and actuators.

MNIST Digit Recognition with Neural Network and Webcam

This project focuses on developing a deep learning model using TensorFlow and Keras to recognize handwritten digits from the MNIST dataset. The model is trained to classify digits (0-9) and is then integrated with a webcam to enable real-time digit recognition. Using OpenCV, the system captures images, preprocesses them, and predicts the handwritten digit using the trained neural network. The project demonstrates the application of artificial intelligence in computer vision and pattern recognition.

Student Marketplace

A dedicated online marketplace for students to buy and sell educational resources such as textbooks, stationery, and other learning materials. The platform ensures a secure and relevant trading environment by implementing restrictions on product types and user registrations. It features an enhanced search functionality to help students quickly find specific products, along with a user-friendly interface for seamless transactions. The goal is to create a cost-effective and convenient solution that simplifies the exchange of educational resources among students.
